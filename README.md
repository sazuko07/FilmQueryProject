# FilmQueryProject
# Description
this program allows a user to search a film database for films, actors, and other search criteria
based on their input (user will be prompted for the correct input)
# Technologies used
Java
SQL
# Lessons Learned
1. Make sure your SQL queries work, before implementing them in java
2. Debugging using println is the best method for break-point checks
3. SQL exceptions are common, and must be dealt with accordingly
4. Interfaces are critical for using SQL in java
5. If the code works in multiple situations, re-use it!
